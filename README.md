📌 Sobre o Projeto

O Projeto Alura Store foi desenvolvido para praticar análise de dados com Python, utilizando visualizações para facilitar a interpretação das informações.

A atividade faz parte da minha formação em Data Science pela Alura, dentro do programa ONE (Oracle Next Education).

🛠️ Ferramentas Utilizadas

Python

Pandas

Matplotlib

Jupyter Notebook

VS Code

🔎 O que foi feito

Durante o projeto foram realizadas:

Exploração e análise do dataset

Criação de gráficos para visualizar os dados

Identificação de padrões e comportamentos

📈 Resultados da Análise

As visualizações permitiram compreender melhor os dados e extrair insights importantes, demonstrando como a análise gráfica auxilia na interpretação de informações.
